System Requirements

Make sure your device is ready to run NextLevel smoothly.

PC Minimum

  • OS: Windows 10 / macOS 11 / Linux
  • CPU: Intel i5-4460 / AMD Ryzen 3
  • RAM: 8GB
  • GPU: GTX 1050 / RX 560
  • Storage: 20GB

PC Recommended

  • OS: Windows 11 / macOS 12
  • CPU: Intel i7-9700K / AMD Ryzen 7
  • RAM: 16GB
  • GPU: RTX 3060 / RX 6700 XT
  • Storage: 20GB SSD

Console / Mobile

  • PlayStation 5, Xbox Series X|S, Nintendo Switch
  • iOS 15+ / Android 10+
  • Stable internet for cloud saves & streaming
